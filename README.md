## 📌 Descrição do Repositório

Este repositório é dedicado ao estudo e aplicação prática de técnicas de Machine Learning, com foco na construção de soluções orientadas a dados.

O projeto contempla todas as etapas do pipeline de dados, incluindo desde a análise exploratória (*Exploratory Data Analysis - EDA*), passando pelo pré-processamento e engenharia de atributos, até o treinamento, validação e avaliação de modelos de aprendizado de máquina.

### 🔍 Abordagens e Técnicas

- Pré-processamento de dados (limpeza, normalização, padronização, encoding)
- Análise exploratória de dados (EDA)
- Engenharia e seleção de features
- Implementação de algoritmos supervisionados e/ou não supervisionados
- Ajuste de hiperparâmetros (*hyperparameter tuning*)
- Avaliação de modelos com métricas como:
  - Accuracy
  - Precision
  - Recall
  - F1-score
- Construção de pipelines de Machine Learning
- Boas práticas de versionamento e reprodutibilidade

### 🛠️ Tecnologias e Ferramentas

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Jupyter Notebook

### 🎯 Objetivo

Este repositório tem como objetivo consolidar conhecimentos em Ciência de Dados e Machine Learning por meio de experimentação prática, servindo também como base para evolução técnica e construção de portfólio.
